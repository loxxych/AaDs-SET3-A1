# AaDS-SET3-A1
Task A1 for SET3 of the HSE SE AaDS 2nd year course

The repository consists of:
- A C++ program calculating the area of the intersection of the 3 given circles using the Monte-Carlo method
- A plot drawing Python script
- Test data set as a .csv file
- Plots
  
The script draws 2 plots:
- The first one illustartes the approximate areas calculated using the C++ program
- The second one shows the relative error
